# Predviđanje srčanih bolesti

## Opis projekta
Projekat se bavi predikcijom prisustva bolesti korišćenjem mašinskog učenja.
Analizirana su dva dataseta: Heart Disease.

## Korišćeni modeli
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Artificial Neural Networks (ANN)

## Tehnologije
- Python
- scikit-learn
- matplotlib / seaborn

## Struktura projekta
- data/ – dataset
- notebooks/ – Jupyter notebook fajlovi
- README.md

## Pokretanje projekta
1. Instalirati potrebne biblioteke
2. Pokrenuti notebook fajlove redom
